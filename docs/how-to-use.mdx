##### 1. Install and run Mutable Web extension

You can find and enable applications in the **Mutable Web browser extension**, which you can install from your browser extension store (for [Google Chrome](https://chromewebstore.google.com/detail/mutable-web/cnahdmdbhkphpbpbjjbfdnmbphbenglc) are currently available).

![Mutable Web of Extension](/img/use_0.png)

You can pin Mutable Web extension button and click on it to open the overlay.

![Mutable Web of Extension](/img/use_1.png)
