### 1. Install and run Mutable Web extension

Find and enable applications in the **Mutable Web** browser extension, which you can install from [**Chrome Web Store**](https://chromewebstore.google.com/detail/mutable-web/cnahdmdbhkphpbpbjjbfdnmbphbenglc) for Google Chrome.

![Mutable Web of Extension](/img/picker/docs-picker-00.png)

After installing and enabling the extension, open the X social network page. Ensure that the **Welcome mutation** by _bos.dapplets.near_ or _bos.dapplets.testnet_, depending on the chosen network, is enabled.

![Welcome mutation on X](/img/picker/docs-picker-01.png)

Pin Mutable Web extension button to have a quick access to its options.

![Mutable Web of Extension](/img/use_0.png)

### 2. Near Social

Try Mutable Web without installing the extension. Open the **[social.dapplets.org](https://social.dapplets.org/)** site (our fork of [Near Social](https://near.social/)).

![Mutable Web of Extension](/img/use_1.png)
