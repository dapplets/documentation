##### 1. Install and run Mutable Web extension

You can find and enable applications in the **Mutable Web browser extension**, which you can install from [**Chrome Web Store**](https://chromewebstore.google.com/detail/mutable-web/cnahdmdbhkphpbpbjjbfdnmbphbenglc) for Google Chrome.

![Mutable Web of Extension](/img/picker/docs-picker-00.png)

You can pin Mutable Web extension button to have a quick access to its options.

![Mutable Web of Extension](/img/use_0.png)
